# TaskFlow — статический планировщик задач

Адаптивный планировщик задач для ПК и телефона.

## Возможности

- Ветки, темы и задачи
- Фильтры: все, сегодня, активные, готовые, важные
- Поиск
- Сортировка
- Редактирование и удаление
- Сохранение в `localStorage`
- Экспорт и импорт JSON-бэкапа
- Работает без сервера и базы данных

## Как залить на GitHub Pages

1. Создай репозиторий на GitHub.
2. Загрузи файлы:
   - `index.html`
   - `styles.css`
   - `app.js`
   - `README.md`
3. Открой `Settings → Pages`.
4. В `Build and deployment` выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохрани.
6. Через минуту GitHub даст ссылку на сайт.

## Как залить на обычный хостинг

Просто загрузи эти файлы в папку сайта, обычно `public_html`, `www` или `htdocs`.

## Важно

Данные хранятся в браузере пользователя. Если открыть сайт на другом устройстве, там будет пусто, пока не импортировать JSON-бэкап.
